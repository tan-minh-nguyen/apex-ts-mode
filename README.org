* apex-ts-mode

Tree-sitter based major mode for Salesforce Apex in Emacs.

** Features

- Syntax highlighting via tree-sitter
- Imenu support for classes, methods, fields
- Indentation support
- Code folding

** Requirements

- Emacs 29.1+ (with tree-sitter support)
- [[https://github.com/aheber/tree-sitter-sfapex][tree-sitter-sfapex]] grammar

** Installation

*** Grammar Installation

#+begin_src emacs-lisp
(add-to-list 'treesit-language-source-alist
             '(apex "https://github.com/aheber/tree-sitter-sfapex" "main" "apex/src"))
(treesit-install-language-grammar 'apex)
#+end_src

*** Package Installation

Add to your load-path:

#+begin_src emacs-lisp
(add-to-list 'load-path "/path/to/apex-ts-mode")
(require 'apex-ts-mode)
#+end_src

** Standalone Usage

The core =apex-ts-mode= works independently without any Salesforce CLI integration:

#+begin_src emacs-lisp
(require 'apex-ts-mode)
;; .cls and .trigger files automatically use apex-ts-mode
#+end_src

** LSP Support (Eglot)

Built-in eglot configuration. Requires Apex LSP JAR (download from Salesforce VSCode extension):

#+begin_src emacs-lisp
;; Set path to Apex LSP JAR
(setq apex-ts-mode-lsp-jar-path "~/.local/lsp/apex-lsp.jar")

;; Register with eglot
(apex-ts-mode-eglot-setup)
#+end_src

** Salesforce Integration (Optional)

For full Salesforce development experience, install [[https://github.com/tan-minh-nguyen/salesforce-minor-mode][salesforce-minor-mode]] and load the extensions:

#+begin_src emacs-lisp
;; Load Salesforce integration
(require 'salesforce-mode)

;; Enable apex-ts-mode extensions
(add-to-list 'load-path "/path/to/apex-ts-mode")
(require 'ob-apex)       ;; Org-babel support (requires salesforce-mode)

(add-to-list 'load-path "/path/to/apex-ts-mode/extensions")
(require 'apex-consult)  ;; Consult navigation
#+end_src

** Extensions

| Extension | Location | Purpose | Requires |
|-----------+----------+---------+----------|
| ob-apex | root | Org-babel for Apex | salesforce-core |
| apex-consult | extensions/ | Consult imenu sources | salesforce-core, consult |
| apex-fold | extensions/ | Code folding | hideshow |
| apex-dap | extensions/ | Debug adapter (WIP) | dap-mode |

** File Associations

#+begin_src emacs-lisp
(add-to-list 'auto-mode-alist '("\\.cls\\'" . apex-ts-mode))
(add-to-list 'auto-mode-alist '("\\.trigger\\'" . apex-ts-mode))
#+end_src
